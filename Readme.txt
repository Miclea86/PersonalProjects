Smart Car Project:

The purpose of this project is to help users to keep track of their vehicle's documents.
This app will help users to never forget when a certain document will expire.

The project was made with python using Django web framework.
The project is a full stack app. powered by Django on the backend and html/css/bootstrap on the font-end.
This app uses SQLite for the database.


The educational purpose of this project is to learn the steps involved in the production of a full stack web app.
